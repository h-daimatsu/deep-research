# 結論セクションの書き方

Deep Research レポートには必ず `## 結論` セクションを置く。エグゼクティブサマリーで代替してはならない。

> **多段推論の再点火（結論の執筆前に適用）**: This task involves multi-step reasoning. Think carefully through the problem before responding. 結論は洞察・対立する見方・調査結果を統合して処方へ落とす工程である。書き出す前に、主題への答え・含意・展望が本文の論理から段階的に導けているかを考え抜く。判断保留や両論併記で逃げず、蓋然性で加重した統合的判断まで推論を進めてから執筆する。

## 役割の違い

| セクション | 役割 | 読み手の位置 |
|---|---|---|
| エグゼクティブサマリー | これから読む前の導入・全体像の提示 | これから読む |
| 結論 | 読み終えた後の総括・含意・行動指針 | 読み終わった |

エグゼクティブサマリーは「これから何を読むか」を示す予告編、結論は「読んだ結果として何を持ち帰るか」を示す集約点。両者を混同しない。

## 独自の洞察・対立する見方との役割分担

| セクション | 役割 |
|---|---|
| 独自の洞察 | 事実を組み合わせた新しい解釈（分析的） |
| 対立する見方 | 評価が分かれる論点の両論併記（論争的） |
| 結論 | それらを踏まえた統合的な判断と含意（統合的・処方的）。「読者が何を持ち帰るべきか」に焦点を絞る |

結論は洞察の再掲ではなく、**洞察と対立する見方を踏まえた上で、読者が行動・判断するための処方を示す**セクション。

## 結論セクションの5要素（必須）

以下の5要素を、箇条書きではなく段落形式の散文で統合的に記述する。

1. **主題への回答**: エグゼクティブサマリーで提示した本レポートの主題に対する最終的な答え
2. **主要な発見の再提示**: 調査で明らかになった事実のうち、最も重要な2〜3点を短く再確認
3. **含意（So what）**: 誰にとって、何を意味するか（顧客・競合・市場・規制当局など関係するステークホルダー別）
4. **推奨事項・注視ポイント**: 読者が取るべきアクション、または注視すべき先行指標
5. **展望**: 短期〜中期で最も蓋然性の高いシナリオ（楽観/悲観の両論併記ではなく、統合的判断としての見通し）

## 判断保留で閉じない

結論セクションは**処方的であるべき**である。以下の結び方は避ける:

- **判断を完全に読者に委ねる結び**: 「どちらとも言えない」「読者の判断に委ねられる」「読者が自らの判断を構成する」で閉じない
- **問いかけで閉じる結び**: 「〜なのか、それとも〜なのか」「今後が注目される」で閉じない
- **両論併記のまま閉じる結び**: 楽観と悲観を並べて終わらせない

情報が不十分で明確な推奨が困難な場合も、以下のように**蓋然性で加重する**:

- 「現時点で利用可能な情報に基づくと、〜シナリオが最も蓋然性が高い」
- 「ただし〜の条件が変われば評価は修正が必要となる」
- 「6〜12ヶ月以内に〜が判明するまでは暫定的な見立てである」

このように限定詞と前提条件を付けることで、判断の責任を引き受けつつ、新情報での修正可能性も残す。判断を避けるために保留するのではなく、**判断した上で前提を明示する**。

## 結論の重点はテーマに応じて動的に決める

本スキルでは固定のレポート型を持たないため、結論で厚く書く要素はテーマの性質に応じて判断する。以下を目安とする:

| テーマの性質 | 結論で特に厚く書く要素 |
|---|---|
| 意思決定を求めるテーマ | 推奨事項（具体的なアクション）、注視ポイント（先行指標） |
| 構造理解を求めるテーマ | 主題への回答（問いへの最終的な答え）、含意（ステークホルダー別影響） |
| 将来予測を求めるテーマ | 展望（蓋然性の高いシナリオ）、推奨事項（備えるべきこと） |
| 比較・選定を求めるテーマ | 推奨事項（選定結論）、含意（選定の代価・トレードオフ） |

ただし5要素すべては必ず含める。比重の違いは「どこを厚くするか」であって「どれを省くか」ではない。

## 書き方のサンプル

```markdown
## 結論

本レポートの主題は「〜」であった。調査の結果、その答えは「〜」である（必要に応じて限定詞を付す）。

{主要な発見の再提示を2〜3点、短い段落で}

この変化の意味は関係者によって異なる。顧客にとっては〜。競合にとっては〜。規制当局にとっては〜。

注視すべき先行指標は次の通りである。{推奨事項・注視ポイントを具体的に}

短期〜中期の展望としては、{最も蓋然性の高いシナリオを1つ、限定詞付きで}。前提条件として〜が維持されることが必要である。
```

**書き方の注意点**:

- 箇条書きではなく段落形式で統合的に記述する（散文で論理を自己完結させる）
- 「本レポートが示すのは」「読者に委ねる」のようなメタ自己言及は避ける
- 問いかけや反語（「〜なのか、それとも〜なのか」）で閉じない
- 文学的比喩（「墓標」「街区」等）で締めない
